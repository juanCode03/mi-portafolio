Proyecto de Gestión de Asistencia

Siga - Sistema de Asistencia Biométrica

Desarrollado por Juan Arévalo

PHP • MySQL • Biometría • Control de asistencia

Descripción del Proyecto:
Aplicación web robusta diseñada para automatizar, auditar y optimizar el control de asistencia del personal de la organización. El sistema reemplaza los métodos tradicionales de registro por una plataforma centralizada que interactúa dinámicamente con las jornadas laborales, permitiendo a los departamentos de Recursos Humanos supervisar en tiempo real los marcajes de entrada, salida, retardos e inasistencias de los empleados.

    🛠️ Stack Tecnológico
  • Backend & Lógica: PHP (Arquitectura MVC)
  • Base de Datos: MySQL / MariaDB (Optimizado con Procedimientos Almacenados y Triggers)
  • Frontend: HTML5, CSS3, JavaScript / jQuery, Bootstrap
  • Servidor / Entorno: Apache / phpMyAdmin
    🚀 Características Clave
  • Arquitectura de Base de Datos Relacional: Diseño e implementación de un esquema e-r óptimo para gestionar entidades complejas como empleados, cargos, departamentos, horarios y el histórico de asistencias.
  • Cálculo Dinámico de Métricas (Tiempo Real): Desarrollo de algoritmos lógicos en el backend que cruzan las horas de entrada permitidas con los marcajes reales para categorizar automáticamente estados de puntualidad, retardos justificados/injustificados y ausencias.
  • Automatización y Pruebas de Carga: Creación de procedimientos almacenados en MySQL para la simulación y poblamiento masivo de registros de asistencia bajo condiciones y calendarios laborales reales (exclusión de fines de semana), garantizando la escalabilidad de las consultas.
  • Dashboard Administrativo: Interfaz intuitiva con reportes estadísticos visuales para la toma de decisiones del personal gerencial.
