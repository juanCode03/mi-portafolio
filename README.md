# Mi Portafolio Personal 💼

Portafolio profesional minimalista enfocado en un diseño limpio, transiciones fluidas y acceso directo a mis proyectos.

## Sitio en Vivo 🌐
Puedes ver el resultado final aquí: [juanCode03.github.io/mi-portafolio/](https://juanCode03.github.io/mi-portafolio/)

## Características Clave ✨
- **Funcionalidad de Contacto Directo:** Integración con *Formspree* para recibir correos de manera eficiente y segura sin necesidad de un backend complejo.
- **Experiencia de Usuario Fluida (UX):** Animaciones dinámicas al hacer scroll con *ScrollReveal* y desplazamiento suave optimizado mediante *Lenis Scroll*.
- **Diseño Totalmente Responsivo:** Adaptación y optimización para una visualización perfecta en dispositivos móviles, tablets y pantallas de escritorio.
- **Enfoque Minimalista:** Estructura directa al grano para facilitar la navegación rápida de los reclutadores.

## Tecnologías e Integraciones 🛠️
| Tecnología / Librería | Uso en el Proyecto |
| :--- | :--- |
| **HTML5** | Estructura semántica del contenido. |
| **CSS3** | Diseño visual, estilos personalizados y adaptabilidad móvil. |
| **JavaScript (ES6+)** | Lógica de interacciones y control de librerías. |
| **ScrollReveal** | Animaciones dinámicas y efectos visuales al desplazar la página. |
| **Lenis Scroll** | Control del desplazamiento suave (*smooth scroll*) para mejorar la UX. |
| **Formspree** | Procesamiento técnico del formulario de contacto. |
